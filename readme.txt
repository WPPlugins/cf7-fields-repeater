Repeat contact form 7 fields and submit them easily.

== Description ==

First Initial Version:

- Requires contact form 7 v4.4 or higher
- Repeat any field you like using jQuery, and submit them.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/contact-form-7-repeater` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. The plugin is functional with one repeater, a license is required to unlock more than one repeater.
